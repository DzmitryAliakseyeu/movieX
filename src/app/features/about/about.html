About project

This project was developed as a final task within the RS School Angular course. It serves as a practical showcase of building a scalable, high-performance web application, demonstrating the core principles of modern frontend development and the power of the Angular ecosystem.
* This product uses the TMDB API but is not endorsed or certified by TMDB
